# BugHunter

Frontend app to manage issues from user experience

## Install

### Step 1. Clone repository

```
git clone https://github.com/kyborq/bug-hunter.git
cd bug-hunter
```

### Step 2. Instal dependencies

```
npm install
```

### Step 3. Configure .env

```
API=12345
```

### Step 4. Run development server

```
npm run dev
```
